# Android Log Data Server

안드로이드 로그 데이터를 관리하기 위한 서버

## Installation
```python
pip install -r requirements.txt
```
* python3.5에서 개발되었음

## Development Stack
* Language : Python
* Database : MongoDB
* Server : Python-Flask

## 사용된 오픈소스
* 서버와 데이터베이스 연결을 위한 pymongo
	* https://github.com/mongodb/mongo-python-driver
